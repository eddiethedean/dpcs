## Reporting a vulnerability

Please report security issues privately to the maintainers. Do not open a public
issue for vulnerabilities that could affect users in production.

Include:

- a description of the issue
- steps to reproduce
- impact assessment, if known
- any suggested fix

We will acknowledge receipt and coordinate a fix and disclosure timeline.
